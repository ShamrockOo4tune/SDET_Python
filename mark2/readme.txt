Выполнил тестовое задание на разработчика SDET / Python.
Для выполнения задания использовался Selenium, паттерн проектирования Page Object.
Технологии Grid и Allure сам ПОКА освоить не смог, но уже разобрался как правильно оформить в виде Pytest.
В папке необходим chromedriver.exe. Скорее всего для корректной работы модулей теста потребуется разместить взамен этого вебдрайвер подходящий к вашему браузеру, подходящей версии.
Тест запускается через модуль tests.py.
Остальные модули импортируются в него.
Подготовительные работы производятся в conftest.py. При запуске через $pytest tests.py, фикстура из conftest.py предварительно отрабатывает автоматически.
При запуке tests.py непосредственно, вводные передаются через конструкцию <if __name__ == '__main__':>
