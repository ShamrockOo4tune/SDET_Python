Выполнил тестовое задание на разработчика SDET / Python.
Для выполнения задания использовался Selenium, паттерн проектирования Page Object.
Технологии Grid и Allure сам освоить не смог, не разобрался как правильно оформить в виде Pytest.
В папке есть chromedriver.exe. Скорее всего для корректной работы модулей теста потребуется разместить взамен этого вебдрайвер подходящий к вашему браузеру, подходящей версии.
Тест запускается через модуль test_send_message.py.
Остальные модули импортируются в него.
